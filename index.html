One Click Copy/Paste

Feature Documentation

Enhance Your Copy/Paste Workflow with Smart Features

📌 Overview

The One Click Copy/Paste Extension simplifies text management, enhances productivity, and streamlines your workflow. With powerful features like quick copy, editable lists, source tracking, and one-click screenshots, managing copied content has never been easier.

✨ Key Features & Functions

  • 🖱️ Right-Click Control: Toggle the Enable/Disable button to manage right-click protection. Enables easy pasting via right-click.
  • 📄 Documentation & Support: Click the doc icon to access detailed documentation and help resources.
  • 📝 Editable Titles: Rename any copied text list item by clicking its title for better organization.
  • 📋 Copy & Paste Functions: Single Left Click to copy text, Single Right Click to paste it instantly.
  • 🗂️ Copy History: All copied text is saved in a list, allowing you to copy from the history.
  • 📌 Latest Copy on Top: Enable this option to keep the most recent copied text at the top of the list.
  • 📤 Export Copied Content: Click the export button to download all copied content.
  • 🔎 Quick Copy & Search: Single Left Click to copy text, Double Left Click to perform a Google search.
  • 🔗 Source Page Tracking: Click the source icon to return to the source page. If the tab is closed, a new one will open.
  • ⚡ Advanced Google Search: Enable this feature to enhance search results with additional filters.
  • ❌ Remove Items: Click the Cross (X) icon to delete individual list items.
  • 📸 Quick Screenshot: Capture screenshots of the active tab with one click.
  • 📂 Download & Manage Screenshots: Save screenshots and use "Clear All" to remove them.

📜 Privacy & Security

  • 🔒 No Tracking, No Data Collection: Your copied text and screenshots remain private on your device.
  • 💡 Fully Local Storage: Data is stored locally, never sent to external servers.
  • ✅ Secure Permissions: Uses only necessary permissions for functionality.

📥 How to Use?

  • 1️⃣ Install the extension from the Chrome Web Store.
  • 2️⃣ Copy any text from a webpage.
  • 3️⃣ Open the extension popup to manage saved text.
  • 4️⃣ Use smart actions like quick copy, Google search, and source tracking.
  • 5️⃣ Capture, download, or clear screenshots anytime.
